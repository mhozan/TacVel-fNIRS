nirs.modules.AR_IRLS

This module runs a GLM using an autoregressive iterative least squares algoroithm for correcting motion and serially correlated error in fNIRS data.
By default, the nirs.modules.GLM module utilizes this (AR-IRLS) algorithm [1].
Example Usage:
raw = nirs.testing.simData;
 
job = nirs.modules.OpticalDensity();
job = nirs.modules.BeerLambertLaw(job);
job = nirs.modules.AR_IRLS(job);
 
Stats = job.run(raw);
